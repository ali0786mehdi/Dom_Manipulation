# 🚀 DOM Mastery Playground

A highly interactive JavaScript project demonstrating advanced DOM manipulation techniques, modern JavaScript features, browser APIs, event-driven architecture, local storage persistence, custom events, dynamic rendering, keyboard shortcuts, and UI state management.

---

## 📸 Project Overview

DOM Mastery Playground is designed as an advanced frontend project that showcases real-world JavaScript concepts frequently used in modern web applications.

Users can:

- Enter their name and receive personalized greetings
- Dynamically interact with generated color boxes
- Track application statistics
- Toggle Dark Mode
- Persist data using Local Storage
- View real-time activity logs
- Trigger custom DOM events
- Use keyboard shortcuts
- Experience debounced input handling

---

# ✨ Features

### Dynamic DOM Creation

Boxes are generated entirely using JavaScript.

```js
document.createElement()
document.createDocumentFragment()
```

---

### Event Delegation

Single listener manages all color boxes.

```js
parent.addEventListener(...)
```

Benefits:

- Better performance
- Cleaner code
- Scalable architecture

---

### Custom Events

Application creates and listens to custom browser events.

```js
new CustomEvent()
```

Example:

```js
document.dispatchEvent(...)
```

---

### Local Storage Persistence

Stores:

- Click Count
- Color Count
- Username
- Theme Preference

Browser API used:

```js
localStorage
```

---

### OOP (Object-Oriented Programming)

State management handled through:

```js
class AppState
```

Encapsulates:

- Data
- Persistence
- Application State

---

### Debouncing

Prevents excessive event execution while typing.

```js
debounce()
```

Used for:

- Search bars
- API requests
- Form validation

---

### Keyboard Shortcuts

Shortcut:

```text
CTRL + D
```

Toggle dark mode instantly.

---

### Real-Time Timer

Uses:

```js
setInterval()
```

Tracks active session duration.

---

### Theme Persistence

Theme preference remains after refresh.

```js
localStorage
```

---

### Activity Logger

Tracks:

- Greetings
- Color Changes
- Theme Switches
- User Actions
